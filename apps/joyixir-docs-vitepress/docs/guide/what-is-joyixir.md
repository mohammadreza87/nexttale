## Tech Stack

Joyixir generates games using:

- **React 18** - UI framework
- **Vite** - Build tool
- **TypeScript** - Type-safe code
- **Tailwind CSS** - Styling
- **shadcn/ui** - UI components
- **Three.js** - 3D graphics (optional)
- **Canvas API** - 2D games
