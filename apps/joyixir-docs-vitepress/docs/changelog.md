# Changelog

All notable changes to Joyixir.

## January 2026

### Jan 3, 2026

#### Split Layout
- **Resizable Panels**: Drag the divider between chat and preview to resize
- **Separate Headers**: Chat and preview panels now have independent headers
- **Chat Header**: Project dropdown with credits, history toggle, copy chat button
- **Preview Header**: View tabs (Preview/Code), device toggles, action buttons
- **Chat History**: Browse and switch between past chat sessions
- **Device Toggles**: Preview your game on Desktop, Tablet, or Mobile sizes

#### Project Management
- **Project Dropdown**: Quick access to dashboard, credits, settings
- **Star Projects**: Mark projects as favorites
- **Rename Projects**: Change project names from the dropdown
- **Remix Projects**: Create a copy of any project

#### Credits System
- **Credits Panel**: Track remaining credits in the chat panel
- **Usage Tracking**: Monitor your credit consumption
- **Add Credits**: Easy access to get more credits

---

## January 2025

### Jan 3, 2025

#### New Modern UI
- **AI Chat Redesign**: New chat interface with avatars, feature checkmarks, and "Next steps" suggestions
- **Suggestion Chips**: Quick action buttons like "Add power-ups", "Sound effects"
- **Credits Display**: See remaining credits in the chat input
- **Header Actions**: Share, Upgrade, and Publish buttons
- **Collapsible Panels**: Code editor and terminal now toggle on/off

#### Performance Improvements
- **Faster Project Loading**: Skip npm install when dependencies already installed
- **In-Session Caching**: Switching projects in same session is now instant
- **IndexedDB Ready**: Infrastructure for cross-session caching

#### AI Improvements
- **Upgraded to Gemini 3 Pro**: Better code generation with 32K output tokens
- **Smart Suggestions**: AI responses now include contextual next steps
- **Feature Extraction**: Bullet points automatically highlighted in responses

---

## December 2024

### Dec 27, 2024

#### Initial Release
- **AI Chat**: Natural language game generation
- **Live Preview**: Real-time game preview with hot reload
- **Code Editor**: Monaco editor with syntax highlighting
- **File Explorer**: Browse and edit project files

- **3D Support**: Automatic Three.js installation
