Joyixir is an **AI-powered game builder** that lets anyone create interactive games and experiences using natural language. Simply describe what you want to build, and watch it come to life in real-time.

## How It Works

<Steps>
  <Step title="Describe Your Game">
    Tell Joyixir what you want to build using natural language. Be specific about gameplay, visuals, and mechanics.
  </Step>
  <Step title="Watch It Build">
    Our AI generates the code in real-time. See the live preview update as files are created.
  </Step>
  <Step title="Iterate & Refine">
    Ask for changes, add features, fix bugs - all through conversation. The AI remembers context.
  </Step>
  <Step title="Export & Deploy">
    Download your game as a ZIP or deploy directly to Vercel, Netlify, or your own server.
  </Step>
</Steps>

## Tech Stack

Joyixir generates production-ready code using modern technologies:

| Technology | Purpose |
|------------|---------|
| **Vite + React** | Fast builds, modern React 18 |
| **TypeScript** | Type-safe code |
| **Tailwind CSS** | Utility-first styling |
| **shadcn/ui** | Beautiful UI components |
| **Three.js** | 3D graphics (when needed) |
| **Canvas API** | 2D game rendering |

## What Can You Build?

- **2D Games**: Platformers, puzzles, arcade games, card games
- **3D Experiences**: Interactive scenes, simple 3D games
- **Interactive Tools**: Calculators, visualizations, dashboards
- **Prototypes**: Quick MVPs and proof-of-concepts

<Note>
  Joyixir runs entirely in your browser using WebContainers. No server needed, no setup required.
</Note>
